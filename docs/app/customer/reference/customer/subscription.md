# Subscription Settings

The Subscription section provides details on the current billing plan, quotas, and invoices. Billing actions are managed via Stripe integration.

<figure><img src="./assets/subscription-content-auto.png" alt="Customer Subscription Management"><figcaption>Subscription management interface.</figcaption></figure>

## Current Subscription

Displays the active plan card (e.g., Enterprise Plan).

- **Quotas**: Limits associated with the plan (e.g., number of surveys per year).
- **Cost**: The yearly or monthly billing rate.
- **Upgrade Or Change Plan**: Action to modify the current subscription tier.
- **Cancel Plan**: Action to terminate the subscription.

## Billing Portal

- **Access Billing Portal**: Redirects to the external Stripe interface to manage payment methods and view detailed billing history.

## Views

The interface is divided into four sub-tabs:

- **Subscription**: Overview of the current plan.
- **Invoice**: History of past invoices and downloadable receipts.
- **Payment**: Payment method configuration.
- **Quota**: Detailed usage metrics against plan limits.
