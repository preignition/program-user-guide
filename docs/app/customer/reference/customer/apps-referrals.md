# Apps and Referrals

These sections allow customers to manage feature extensions and referrals.

## Apps

The Apps tab displays available feature extensions.

- **Status**: Indicates if an app is activated or deactivated for the customer account.
- **Activation**: Action to enable an app. *Note: Adding new apps may affect subscription quotas.*

## Referrals

The Referral tab provides mechanisms to refer other organizations.

- **Referral Tools**: Interfaces to generate referral links or view rewards and discounts earned.
