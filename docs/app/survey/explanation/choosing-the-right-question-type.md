# Choosing the Right Question Type

Selecting the appropriate question type is crucial for ensuring a smooth respondent experience and collecting high-quality, actionable data. This document provides an overview of the available question types in Accessible Surveys and offers guidance on when to use each one.

## Text Inputs

These types are used when you need the respondent to provide information in their own words.

* **Text Field**: Use this for short, single-line answers like names, job titles, or short phrases.
* **Text Area**: Best for longer responses, such as comments, feedback, or detailed descriptions. It allows multiple lines of text and can be configured to auto-adjust its height.

## Specific Format Inputs

These specialized text fields include built-in validation to ensure the data is entered in the correct format.

* **Number**: Restricts input to numeric values. Use this for ages, quantities, or counts.
* **Email**: Ensures the respondent enters a valid email address format.
* **Telephone (Tel)**: Designed for phone numbers.
* **Date**: Provides a date picker for selecting specific calendar dates.
* **URL**: Validates that the input is a correctly formatted website address.

## Selection Inputs

Selection questions are used when you want respondents to choose from a predefined list of options.

* **Radio**: Use when you want the respondent to select **exactly one** option from a small list (usually 2-5 items). All options are visible at once.
* **Checkbox**: Use when the respondent can select **one or more** options from a list.
* **Dropdown**: Best for selecting **one** option from a long list (e.g., choosing a country). It saves space by hiding the options until clicked.
* **Switch**: A binary toggle (On/Off or Yes/No). Use this for simple preference settings.
* **Check**: A selection list where users select an option. While similar to Radio, it may provide a different visual representation.

## Advanced Evaluation

These types help you understand preferences, satisfaction levels, or priorities.

* **Rating**: Allows respondents to rate something on a visual scale (e.g., a star rating). Use this to measure satisfaction or quality.
* **Ranking**: Enables respondents to drag and drop items into their preferred order. Use this to determine priorities or preferences among several choices.

## Media Uploads

Use these when you need the respondent to provide external files or recordings.

* **File**: A general-purpose upload field for documents or other file types.
* **Image**: Specifically for uploading image files (JPG, PNG, etc.).
* **Audio**: For uploading audio recordings.
* **Video**: For uploading video files.

{% hint style="info" %}
**Pro Tip**: When using selection inputs, consider the number of options. If you have more than 7-10 options for a single-choice question, a **Dropdown** is usually better for accessibility and readability than a long list of **Radio** buttons.
{% endhint %}
