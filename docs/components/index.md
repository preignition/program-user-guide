# Shared Components

Many features within the Accessible Data platform are built using shared, reusable components. This ensures consistency in user experience, accessibility standards, and design across all applications (like the Survey App, Customer App, and User App).

Browse the documentation for our shared components below:

* [**Rich Text Editor**](./md-editor.md) - The powerful, Markdown-based text editor used for rich content authoring, supporting accessibility themes, responsive layouts, and interactive widgets.
