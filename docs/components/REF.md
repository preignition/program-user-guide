# Reference

- [lit-app/shared/md/parse.ts](../../accessibleData/lit-app/shared/md/parse.ts): Utility for parsing Markdown and sanitizing the resulting HTML.

- [app/app-base/public/variables.css](../../accessibleData/app/app-base/public/variables.css): Global CSS variables controlling the display of theme-aware content.
