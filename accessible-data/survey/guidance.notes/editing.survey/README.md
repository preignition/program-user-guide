# Editing a Survey

This section provides detailed guidance on all aspects of editing a survey.

Whether you are creating a new survey or refining an existing one, these notes will help you create effective and engaging surveys.

## Core Topics

We cover the following essential areas:

* **Survey Builder Interface**: An overview of the survey builder interface and its key components.

* **Survey Structure**: Understanding how surveys are structured using Pages, Sections and Questions.

* **Introduction Page**: How to write an engaging introduction for your survey.

* **Thank You Page**: How to write a thank you message for your survey.

* **Submit Message**: How submit messages work, if you choose to use them.

* **Adding Content to a Form**: Exploring all the different options for adding questions and content to your form

* **Testing a Form**: How to test your form as you build it.

* **Using Form Logic**: Using logic to create dynamic and personalized survey paths.  Basic logic is available for free users, but advanced logic is available for paid users.

* **Using Tooltips**: How to use tooltips to explain difficult words or technical terms.

## Advanced Features

Guidance Notes are also available for the following advanced features which are available for certain plans:

* **Using the Image Library**: Managing and inserting images into your survey.

* **Viewing Prompts**: Accessing and reviewing AI prompts.

* **Restoring Deleted Items**: How to recover items that have been accidentally removed.
